# Windows 12 Applications Suite

Welcome to the apps folder of the system. This directory contains the individual applications and utility modules that run within the **Windows 12** environment.

## Included Modules
- **WinBrowser:** A custom-styled web-rendering module.
- **Terminal:** The command-line interface for system configuration.
- **CoreSearch:** The primary indexer for the Windows 12 ecosystem.

## Developing for Windows 12
To add a new application:
1. Create a new subdirectory within `/apps`.
2. Ensure your styles follow the "Architect" design standard (Glassmorphism, Purple/Magenta accents).
3. Register the app in the main system manifest to ensure the launcher detects it.
