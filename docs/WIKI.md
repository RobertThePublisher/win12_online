# 🌌 Overview

WIN12 WEB is a futuristic web operating system built using:

* HTML5

* JavaScript
* GitHub Pages

The project focuses on:

* modular UI systems
* neon-inspired visuals
* browser desktop environments
* scalable architecture
* experimental operating system concepts

# 🎨 CSS Architecture

The CSS system is modularized into separated files.

Example:

```txt
css/
├── win12.css
├── browser.css
├── desktop.css
├── taskbar.css
├── windows.css
├── animations.css
└── themes.css
```

Purpose:

* easier maintenance
* scalable architecture
* contributor-friendly structure
* cleaner debugging
