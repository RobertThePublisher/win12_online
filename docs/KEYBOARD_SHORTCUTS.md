# Keyboard Shortcuts — Windows 12 Conceptual Mock-Up

This document outlines all keyboard shortcuts available in this **conceptual mock-up** of Windows 12.

## System Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Esc` | Open Start Menu |
| `Ctrl+L` | Lock Screen |
| `F5` | Refresh Desktop |
| `Ctrl+Shift+/` | Show Keyboard Shortcuts Help |

## Application Shortcuts

| Shortcut | Application |
|----------|-------------|
| `Ctrl+Shift+E` | File Manager |
| `Ctrl+Shift+B` | Browser (WinBrowser) |
| `Ctrl+Alt+T` | Terminal |
| `Ctrl+Shift+N` | Notepad (Text Editor) |
| `Ctrl+Shift+C` | Calculator |
| `Ctrl+Shift+I` | Settings |
| `Ctrl+Shift+P` | Paint |
| `Ctrl+Shift+D` | Calendar |
| `Ctrl+Shift+M` | Media Player |
| `Ctrl+Shift+W` | Weather |
| `Ctrl+Shift+K` | Clock / World Time |

## Window Management

| Shortcut | Action |
|----------|--------|
| `Alt+F4` | Close Focused Window |
| `Ctrl+Shift+↑` | Maximize Window |
| `Ctrl+Shift+↓` | Minimize Window |

## Additional Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+S` | New Sticky Note |

---

## Notes

- This is a **conceptual mock-up** of Windows 12, not an official product
- All keyboard shortcuts are intercepted and mapped to in-app functionality
- Browser developer tools can still be accessed via `F12` or right-click → Inspect
- Shortcuts persist and are configurable through the Settings application
- Keyboard bindings are stored in browser `localStorage`
