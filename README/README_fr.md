### Aperçu du Projet

Une simulation d'interface utilisateur basée sur le web incluant :

* Barre des tâches centrée et menu Démarrer.
* Esthétique "Architect" personnalisée (Glassmorphism / Mode Sombre).
* Applications prototypes fonctionnelles.
