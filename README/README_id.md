# Windows 12 Concept Mock-up

**Penyangkalan:** Proyek ini adalah mock-up konsep tidak resmi. Tidak dibuat, didukung, atau disetujui oleh Microsoft. Ini adalah proyek pribadi untuk tujuan pendidikan dan eksplorasi desain.

### ⚠️ Peringatan Hukum

* **Tidak Ada Afiliasi:** Sistem ini tidak berafiliasi dengan Microsoft dan tidak mengklaim mewakilinya.
* **Larangan:** Segala impersonasi tidak sah atau unggah ulang proyek ini tanpa izin eksplisit dapat mengakibatkan permintaan penghapusan DMCA.
* **Tanggung Jawab:** Saya tidak bertanggung jawab atas penyalahgunaan konsep ini, atau klaim "sistem operasi defensif palsu" oleh pihak ketiga. Ini murni mock-up konsep antarmuka.

### Ikhtisar Proyek

Simulasi antarmuka pengguna berbasis web yang mencakup:

* Taskbar tengah dan Start Menu.
* Estetika "Architect" khusus (Glassmorphism / Dark Mode).
* Aplikasi prototipe yang fungsional.
