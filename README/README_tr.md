# Windows 12 Konsept Maketi

**Yasal Uyarı:** Bu proje resmi olmayan bir konsept maketidir ve Microsoft ile bağlantılı değildir. Microsoft tarafından oluşturulmamış, onaylanmamış veya desteklenmemiştir. Bu, eğitim ve tasarım amaçlı kişisel bir projedir.

---

### ⚠️ Yasal Uyarı
* **Yasal Uyarı:** Bu işletim sistemi Microsoft ile bağlantılı değildir ve Microsoft'u temsil ettiğini iddia etmemektedir.

* **Yasal Uyarı:** Bu projenin açık izin olmadan yetkisiz kullanımı veya yeniden yüklenmesi, DMCA kaldırma talebine yol açabilir.

* **Yasal Uyarı:** Bu konseptin herhangi bir kötüye kullanımından veya üçüncü tarafların "sahte güvenlik işletim sistemi" iddialarından sorumlu değilim. Bu sadece kavramsal bir kullanıcı arayüzü maketidir.

---

### Proje Genel Bakışı
Web tabanlı kullanıcı arayüzü simülasyonu, şunları içerir:
* Ortalanmış görev çubuğu ve Başlat menüsü.

* "Mimari" kullanıcı estetiği (glassmorphism/karanlık mod).

* İşlevsel uygulama prototipleri.
