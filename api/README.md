# WIN12 WEB API

> Internal API systems for WIN12 WEB.

The `api/` folder contains internal browser operating system APIs used by:

* applications
* window systems
* widgets
* services
* desktop integrations

---

# 🌌 Purpose

This folder helps separate:

* UI systems
* backend-like browser logic
* modular internal communication

The goal is scalable architecture while keeping the project organized.

---

# 📁 Files

| File                | Purpose                         |
| ------------------- | ------------------------------- |
| `system.js`         | Core operating system functions |
| `apps.js`           | App registration and management |
| `window-manager.js` | Window handling APIs            |
