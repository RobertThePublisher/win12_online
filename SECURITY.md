# Security Policy

## Supported Versions

Only the latest version of Win12 Online is actively supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Preferred method:**  
Email: [tangtanghasan@gmail.com](mailto:tangtanghasan@gmail.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

We will acknowledge your report within 48 hours and aim to address valid security issues as quickly as possible.

## Note

This is a front-end concept project (HTML + CSS + JS). Most "security" issues will be related to client-side behavior rather than server vulnerabilities.

Thank you for helping keep the project safe.
