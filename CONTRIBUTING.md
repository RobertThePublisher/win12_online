# Contributing to Windows 12 Concept

Thank you for your interest in contributing to **Win12 Online**!

This is a creative front-end concept project with an "Architect" glassmorphism aesthetic. We welcome contributions from developers, designers, translators, and enthusiasts.

## How You Can Contribute

- Report bugs or suggest new features
- Add or improve mock applications
- Improve Electron desktop version
- Create or enhance translations
- UI/UX improvements and animations
- Documentation updates

---

## Contribution Guidelines

Fork the repository and create a new branch (feature/amazing-idea or fix/bug-name)
Make your changes while keeping the Architect visual style (glassmorphism, dark theme, purple accents)
Test in both browser and Electron
Submit a Pull Request with a clear description and screenshots (if UI-related)

Code Style

Modern JavaScript (ES6+)
Clean, readable code
Consistent with existing structure

Questions?
Feel free to open an Issue or contact me directly: tangtanghasan@gmail.com

Note: This project is an unofficial concept mock-up and has no affiliation with Microsoft.
